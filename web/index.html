How it works

Live TxLINE input

The runner polls the real 1X2 odds; the home-win probability is the price both agents see each tick.

Deterministic strategies

Fixed-point, no floats, all saturating — so every decision replays bit-identically on Solana.

Trustless settlement

Dispute a tick and the chain replays it via the tickpruv referee; the loser's bond is slashed.

Built on tickpruv · PRUV Network — don't trust, verify. devnet · illustrative replay · live mode drives this from real TxLINE odds
